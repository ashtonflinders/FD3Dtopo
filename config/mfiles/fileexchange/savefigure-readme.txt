
savefigure was designed for matlab 7 and above. Included in the package is an unsupported version that might work in older matlab versions (but does not support auto margins). 


I hope you will find it useful.



savefigure package contents file contents:
-----------------------------------------------------------------
savefigure.m            The matlab function for saving figures. Requires Matlab 7+

savefigure.m.old        A unsupported version that should work in older Matlab versions

parseArgs.m             A helper function required by savefigure.m
